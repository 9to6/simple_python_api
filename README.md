# simple_python_api

simple python django Rest api server



Docker Usage
-----------

<ul>
  <li>
  Build<br>
  ```console
  $ sudo docker build -t simple_api:0.0 .
  ```
  </li>
  <li>
  Start App<br>
  ```console
  $ sudo docker run -p 8000:8000 -i -t --name simple_api simple_api:0.0
  ```
  </li>
</ul>
